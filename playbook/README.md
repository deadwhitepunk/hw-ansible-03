### README PLAYBOOK
Данный playbook устанавливает Clickhouse, Nginx, Vector, Lighthouse

#### Clickhouse
1. Скачивается дистрибутив.
2. Устанавливаются пакеты и обновляется apt кэш
3. Вызывается хэндлер на запуск clickhouse
4. Для проверки ожидается старт clickhouse сервера по порту 9000
5. Создается база данных

#### Vector
1. Скачивается дистрибутив.
2. Устанавливаются пакеты и обновляется apt кэш
3. Дэплоится шаблон конфигурации
4. Вызывается хэндлер на запуск Vector

#### Lighthouse
1. Устанавливаем необходимые пакеты, а именно git и nginx и обновляется apt кэш
2. Клонируем репозиторий
3. Устанавливаем права на директорию
4. Дэплоится шаблон конфигурации Lighthouse на базе nginx
5. Переносится сайт в sites-eneabled и перезапускается nginx
6. Проверка на запуск сервиса nginx

#### Vars
В проекте для каждого сервиса есть своя директория для настройки переменных.

#### Тэги

#### Запуск проекта
В директории проекта запускаем `ansible-playbook -i ./playbook/inventory/prod.yml ./playbook/site.yml`
